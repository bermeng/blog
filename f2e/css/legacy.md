# 传统盒模型

## 盒模型

- 内容
- 内边距
- 边框
- 外边距

## 外边距叠加

- 垂直相邻的外边距会折叠
    - 合并规则
        - 取两者中较大值
        - 有一个为负值：正 - 负
        - 两个都为负值：负 + 负
- 父子元素，无**内边距，边框**
- 拥有外边距的空元素，无**内边距和边框**

## 三种基本定位机制

- 标准文档流 
    - 块级元素和行级元素组成
    - 从上到下，从左到右
- 浮动
    - 脱离文档流
    - 设置``float``属性
- 绝对定位
    - 脱离文档流
    - 设置``position``属性
        - ``static``
        - ``relative``
        - ``absolute``
        - ``fixed``
